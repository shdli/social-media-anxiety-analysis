# Digital  Wellbeing & Mental Health Analysis
## Project Overview
This project investigates the relationship between digital usage habits and mental wellbeing. 
By analyzing a dataset of over 6,000 participants, we aim to uncover how screen time, notifications, 
and sleep patterns correlate with anxiety levels.
#Technologies Used
Python

Pandas (for data analysis)

Matplotlib / Seaborn (for data visualization)

Jupyter Notebook
